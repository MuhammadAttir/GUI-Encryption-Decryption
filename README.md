# 🔐 Cipher Encryption App

A simple Windows GUI application built in C++ using WinAPI that allows users to encrypt and decrypt text using classical cryptographic techniques.

---

## ✨ Features
- 🔑 Caesar Cipher (Encrypt & Decrypt)
- 🔐 Vigenere Cipher (Encrypt & Decrypt)
- 🖥️ Simple and user-friendly GUI
- ⚡ Fast text processing
- 🧠 Input validation for shift and key

---

## 🛠️ Technologies Used
- C++
- WinAPI (Windows GUI programming)

---

## 🚀 How to Run
1. Compile the program using a C++ compiler (e.g., MinGW)
2. Run the generated `.exe` file
3. Enter text in the input box
4. Choose cipher method:
   - Caesar → enter shift value
   - Vigenere → enter key
5. Click **Encrypt** or **Decrypt**

---

## 📷 Interface Overview
- Input text area
- Cipher selection (Radio buttons)
- Shift / Key input
- Encrypt & Decrypt buttons
- Output display

---

## 📌 Notes
- Caesar cipher requires a numeric shift value
- Vigenere cipher requires alphabetic key only
- Non-alphabet characters remain unchanged
